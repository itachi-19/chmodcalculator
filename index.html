Chmod Calculator

Provide a decimal number in the range 0,777 (composed of octal numbers) to calculate the resulting permissions string. In Unix and Unix-like operating systems, chmod is the command and system call which is used to change the access permissions of file system objects (files and directories). It is also used to change special mode flags. The request is filtered by the umask. The name is an abbreviation of change mode. The three rightmost digits define permissions for the file owner, the group, and others.
